# 📊 IBM Telco Customer Churn Prediction

An end-to-end Machine Learning project focused on predicting customer churn in the telecommunications industry.  
The goal of this project is to identify customers at risk of leaving and generate actionable business insights to improve customer retention.

---

# 📌 Project Overview

Customer churn is one of the biggest challenges for telecom companies. Losing customers directly impacts revenue, so predicting churn can help businesses take preventive actions.

In this project, we analyze the **IBM Telco Customer Churn dataset** and build a machine learning model to predict whether a customer is likely to leave the company.

---

# 📂 Dataset

The dataset used in this project is the **IBM Telco Customer Churn dataset**, which contains information about telecom customers including:

- Customer demographics
- Account information
- Services used
- Contract type
- Monthly charges
- Churn status

Dataset file included in this repository.

---

# 🔎 Project Workflow

1. **Data Cleaning**
   - Handled missing values
   - Converted data types
   - Removed inconsistencies in the dataset

2. **Exploratory Data Analysis (EDA)**
   - Analyzed churn distribution
   - Investigated the relationship between churn and customer features
   - Visualized patterns using charts and plots

3. **Feature Engineering**
   - Encoded categorical variables
   - Prepared data for machine learning models

4. **Model Building**
   - Implemented a **Random Forest Classifier**
   - Split data into training and testing sets (80/20)

5. **Model Evaluation**
   - Accuracy score
   - Confusion Matrix
   - ROC-AUC analysis

---

# 📈 Results

- Model: **Random Forest Classifier**
- Accuracy: **~79%**
- ROC-AUC Score: **~0.84**

The model demonstrates a strong ability to distinguish between customers who will churn and those who will stay.

---

# 💡 Business Insights

Key findings from the analysis:

- Customers with **month-to-month contracts** have the highest churn rate.
- **Fiber optic internet users** show higher churn compared to other services.
- Customers without **technical support** are more likely to leave.

### Recommendations

- Encourage long-term contracts with discounts or loyalty rewards.
- Improve onboarding and support for fiber internet users.
- Offer technical support packages to high-risk customers.
